# 梦幻星空隧道

纯 Canvas 实现的 3D 星空隧道效果，彩色星星从画面中心向四周飞散。

## 效果特点

- 800 个 3D 粒子，6 种星星形状（实心星、空心星、十字、光晕、闪光、圆点）
- 6 种颜色：青色、紫色、粉色、天蓝、白色、淡紫
- 3D 透视投影，星星从远处飞向镜头
- 闪烁发光效果
- 自适应全屏
